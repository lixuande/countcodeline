# countcodeline
统计路径下的代码行数

``` 

git clone https://github.com/lixuande/countcodeline.git
cd countcodeline
sh countcodeline.sh xxxx

```

统计的是xxx路径下的所有h、hpp、c、cpp代码行数


# deletelogfile
删除路径下的.log结尾的文件

``` 

git clone https://github.com/lixuande/countcodeline.git
cd countcodeline
sh deletelogfile.sh xxxx

```

删除的是xxx路径下的所有.log结尾的文件
